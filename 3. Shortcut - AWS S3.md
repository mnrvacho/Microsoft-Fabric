# 3. Fabric Shortcut으로 AWS S3 데이터 가져오기

이 가이드는 **Microsoft Fabric**의 **Shortcut** 기능으로 **AWS S3**의 데이터를 **Lakehouse**에 가져옵니다. 


<br/>

1. `Create` -> `Lakehouse` -> lakehouse 이름 입력 -> `Create`
